Bienvenue dans la documentation de l'API RiDERgy. Ce site documente les API
que les partenaires et intégrateurs peuvent utiliser pour interagir avec la
plateforme RiDERgy.

## API disponibles

<CardGroup cols={1}>
  <Card title="Energy Management API" icon="bolt" href="/fr/api-reference/energy-management/health">
    Lisez et définissez par programmation les limites de puissance de charge
    pour les sites, les bornes individuelles, les organisations et les
    clusters multi-sites.
  </Card>
</CardGroup>

## Pour commencer

1. Consultez [Authentification](/fr/authentication) pour apprendre comment
   obtenir et utiliser une clé API.
2. Consultez le [Modèle de priorité](/fr/concepts/priority-model) pour
   comprendre comment les profils de limites sont planifiés et résolus.
3. Utilisez l'onglet **Référence API** pour tester directement les endpoints
   sur l'API en production.
