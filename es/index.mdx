Bienvenido a la documentación de la API de RiDERgy. Este sitio documenta las APIs que
los partners e integradores pueden usar para interactuar con la plataforma RiDERgy.

## APIs disponibles

<CardGroup cols={1}>
  <Card title="Energy Management API" icon="bolt" href="/es/api-reference/energy-management/health">
    Lee y configura de forma programática los límites de potencia de carga para ubicaciones,
    cargadores individuales, organizaciones y clústeres entre ubicaciones.
  </Card>
</CardGroup>

## Primeros pasos

1. Lee [Autenticación](/es/authentication) para aprender cómo obtener y usar una
   clave de API.
2. Revisa el [Modelo de prioridad](/es/concepts/priority-model) para entender cómo
   se programan y resuelven los perfiles de límite.
3. Usa la pestaña **API Reference** para probar los endpoints directamente contra la
   API en producción.
